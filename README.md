## Описание программы

Random Task Generator — это простое графическое приложение (GUI), которое помогает генерировать случайные задачи из предопределённого списка. Приложение позволяет:
* генерировать случайную задачу по нажатию кнопки;
* просматривать историю сгенерированных задач;
* фильтровать историю по типам задач (учёба, работа, спорт, отдых);
* добавлять новые задачи в список с указанием типа;
* сохранять историю задач между запусками приложения (в формате JSON).

## Функциональность

* **Генерация случайной задачи.** При нажатии на кнопку «Сгенерировать задачу» приложение выбирает случайную задачу из списка и отображает её в главном окне.
* **История задач.** Все задачи из списка отображаются в виде прокручиваемого списка с указанием типа.
* **Фильтрация.** Пользователь может фильтровать отображаемую историю по типу задачи с помощью выпадающего списка (варианты: «Все», «учёба», «работа», «спорт», «отдых»).
* **Добавление новых задач.** В нижней части окна есть поля для ввода текста новой задачи и выбора её типа. При нажатии кнопки «Добавить в список» задача добавляется в общий список и сохраняется в файле `tasks.json`.
* **Валидация ввода.** Приложение проверяет, что текст новой задачи не является пустой строкой. В случае ошибки выводится сообщение.
* **Сохранение данных.** Все задачи (включая добавленные пользователем) сохраняются в файле `tasks.json` и загружаются при следующем запуске приложения.

## Примеры использования

1. Запуск приложения:
* Запустите main.py. Появится окно приложения.
* Изначально в истории будут отображаться предопределённые задачи.

2. Генерация задачи:
* Нажмите кнопку «Сгенерировать задачу». В верхней части окна появится случайная задача из списка.

3. Фильтрация истории:
* Выберите тип задачи в выпадающем списке «Фильтр по типу». Список истории обновится, отображая только задачи выбранного типа.

4. Добавление новой задачи:
* Введите текст задачи в поле «Новая задача:».
* Выберите тип задачи из выпадающего списка.
* Нажмите «Добавить в список». Задача появится в общем списке истории.

5. Проверка сохранения:
* Закройте приложение.
* Снова запустите main.py. Вы увидите, что все добавленные задачи сохранились в истории.
